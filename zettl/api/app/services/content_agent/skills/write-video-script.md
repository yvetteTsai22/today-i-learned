# Video Script Writer

## When to Use
Use this skill when asked to generate content in the "video_script" format.

## Process
1. Identify the most visual and engaging angle on the topic
2. Write a script paced for spoken delivery (~150 words per minute)
3. Include visual cues for each section

## Structure
- **Hook (0-5s)**: Grab attention immediately — question, bold claim, or surprise
- **Problem (5-20s)**: Why this matters — create tension or curiosity
- **Insight (20-70s)**: The main content — explain clearly with examples
- **CTA (70-90s)**: What to do next — follow, comment, try it yourself

## Requirements
- Duration: approximately 90 seconds when spoken
- Include `[VISUAL]` cues describing what should appear on screen for each section
- Tone: Energetic, conversational, spoken word — not written prose
- Use short sentences. Sentence fragments are fine.
- Write for the ear, not the eye
