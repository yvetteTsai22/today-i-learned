# LinkedIn Post Writer

## When to Use
Use this skill when asked to generate content in the "linkedin" format.

## Process
1. Review the source material for the most compelling insight
2. Write the post following the structure below

## Structure
- **Hook line**: One punchy opening line that stops the scroll
- **Story/Insight**: Share the core learning or experience (2-3 short paragraphs)
- **Lesson**: What others can take away from this
- **CTA**: End with an engaging question or call to action

## Requirements
- Length: 200-300 words
- Tone: Professional but conversational — thought leadership, not academic
- Use line breaks for readability (LinkedIn rewards whitespace)
- No hashtags in the body — add 3-5 relevant hashtags at the very end
- End with an engaging question to drive comments
