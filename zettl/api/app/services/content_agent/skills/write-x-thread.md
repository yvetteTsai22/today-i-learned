# X Thread Writer

## When to Use
Use this skill when asked to generate content in the "x_thread" format.

## Process
1. Identify the single most shareable insight from the source material
2. Break it into a narrative arc across 5-7 tweets
3. Write each tweet to stand alone while building on the previous

## Structure
- **Tweet 1**: Strong hook with thread emoji — make people want to read more
- **Tweets 2-5**: Core content — one idea per tweet, building a narrative
- **Tweet 6-7**: Summary + call to action (follow, repost, reply)

## Requirements
- 5-7 tweets total
- Each tweet: strictly under 280 characters
- Format each tweet on its own line with tweet number (1/, 2/, etc.)
- Tone: Punchy, value-dense, conversational
- First tweet must hook — no "Thread:" or "A thread about..." openers
- Last tweet: clear CTA
